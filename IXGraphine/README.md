### Installation

IXGrapine requires [Spark](https://github.com/HUSKI3/Spark/) to run.

Install the package and run it!

```sh
$ spark install ixgraphine
$ ixgraphine
```

To run from raw
```sh
$ chmod +x ixgraphine && ./ixgraphine 
```
